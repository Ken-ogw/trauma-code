AIS（Abbreviated Injury Scale）
各部位の損傷を0〜6で評価する重症度スケール。
スコア重症度意味
0損傷なし—
1軽傷Minor
2中等症Moderate
3重症Serious
4重篤Severe
5最重症Critical
6致死的Unsurvivable
ISS（Injury Severity Score）
6部位のAISスコアのうち、上位3部位の二乗和。AIS=6がひとつでもあれば自動的にISS=75。
ISS = AIS₁² + AIS₂² + AIS₃²
ISS重症度分類
1〜8軽傷（Minor）
9〜15中等症（Moderate）
16〜24重症（Serious）
25〜39重篤（Severe）
40〜74最重症（Critical）
75致死的（Unsurvivable）
RTS（Revised Trauma Score）
GCS・収縮期血圧・呼吸数をコード化して重み付け加算。最大値は7.84。
RTS = 0.9368×GCScode + 0.7326×SBPcode + 0.2908×RRcode

Ps（TRISS法・生存確率）
RTS・ISS・年齢・受傷機転からロジスティック回帰で生存確率を算出。Boyd CR et al. 1987に基づく。
Ps = 1 / (1 + e−b)
b = b₀ + RTS係数×RTS + ISS係数×ISS + 年齢補正
